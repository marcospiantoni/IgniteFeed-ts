# Ignite Feed

## Projeto do curso da Rocketseat - trilha Ignite -  React.

### Funcionalidades: 
+ Links no post funcionando. 
+ Comentar e deletar comentário.
+ Dar like ou deslike no comentário.

### Projeto feito com: 
+ React + Vite 
+ Typescript
